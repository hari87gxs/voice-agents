Here's how you can request for a GXS FlexiCard fee waiver:
Via GXS FlexiCard transaction history
Log in to the GXS Bank app.
Navigate to your GXS FlexiCard transaction history.
Select the transaction for which you want a fee waiver.
Tap on 'Get help'.
Select ‘Requesting a fee waiver’ button and complete the form.

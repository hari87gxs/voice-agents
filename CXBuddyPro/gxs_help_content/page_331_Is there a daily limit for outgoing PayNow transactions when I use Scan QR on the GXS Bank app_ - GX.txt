================================================================================

Yes, there is a daily limit for PayNow transactions using the Scan QR feature on the GXS Bank app. This limit is connected to your GXS Biz Account's daily transfer limit, which is set at a default of S$10,000. If you need to change this limit, you can do so by following these steps:
1. Go to 'Profile'.
2. Tap on 'Change Limits'.
3. Select 'Daily Transfer Limit'.
